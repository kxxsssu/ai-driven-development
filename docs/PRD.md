# PRD: 커뮤니티 기반 AI 이미지 생성 플랫폼

## 1. 제품 개요

### 1-1. 제품명 (가칭)

**CanvasHub**
AI 이미지 생성과 커뮤니티 기능을 결합한 창작 플랫폼

---

### 1-2. 제품 목적

사용자가 AI를 활용해 이미지를 생성하고, 생성한 결과물을 개인 갤러리에서 관리하며, 다른 사용자들과 공유하고 상호작용할 수 있는 커뮤니티 중심 플랫폼을 제공한다.

단순한 AI 이미지 생성 도구를 넘어, 사용자들이 서로의 작품과 프롬프트를 탐색하며 영감을 얻고 새로운 콘텐츠를 만들어내는 창작 생태계를 구축하는 것이 목표다.

---

### 1-3. 개발 배경

최근 AI 이미지 생성 기술의 발전으로 누구나 쉽게 고품질 이미지를 제작할 수 있게 되었다. 그러나 현재 대부분의 서비스는 다음과 같은 한계를 가진다.

* 생성 결과물을 체계적으로 관리하기 어렵다.
* 커뮤니티 기능이 제한적이다.
* 다른 사용자의 창작물을 참고하거나 재활용하기 어렵다.
* 이미지 생성과 공유 경험이 분리되어 있다.

이 서비스는 다음 흐름을 하나의 경험으로 연결한다.

> 생성 → 저장 → 공개 → 탐색 → 상호작용 → 재창작

이를 통해 사용자 참여와 콘텐츠 생산이 지속적으로 순환하는 플랫폼을 구축한다.

---

### 1-4. 해결하려는 문제

| 문제         | 설명                          |
| ---------- | --------------------------- |
| 생성물 관리 어려움 | 생성한 이미지가 누적되면서 정리와 탐색이 어려움  |
| 공유 경험 부족   | 결과물을 다른 사용자와 쉽게 공유하기 어려움    |
| 영감 탐색 제한   | 다른 사용자의 프롬프트와 스타일을 탐색하기 어려움 |
| 커뮤니티 부재    | 창작 중심의 소통과 피드백 경험이 부족함      |

---

### 1-5. 핵심 가치 제안

* 누구나 쉽게 AI 이미지를 생성 가능
* 생성 결과를 개인 갤러리에서 체계적으로 관리 가능
* 커뮤니티를 통한 작품 공유 및 반응 확인 가능
* 다른 사용자의 프롬프트와 스타일을 참고 가능
* 창작 활동이 지속적으로 이어지는 플랫폼 경험 제공

---

# 2. 대상 사용자

## 2-1. 일반 사용자

AI 이미지 생성에 관심이 있으며, 간단한 텍스트 입력만으로 원하는 이미지를 만들고 싶어 한다. 생성한 이미지를 저장하고 공유하는 경험을 원한다.

### 주요 니즈

* 쉬운 이미지 생성
* 간단한 UI
* 빠른 결과 확인
* SNS 스타일 공유 경험

---

## 2-2. 디자이너 및 크리에이터

아이디어 스케치, 콘셉트 작업, 콘텐츠 제작을 위해 AI 이미지를 활용한다. 다양한 스타일과 프롬프트를 실험하고 관리할 필요가 있다.

### 주요 니즈

* 생성 결과 관리
* 프롬프트 반복 활용
* 스타일 참고
* 다른 사용자 작품 탐색

---

## 2-3. 커뮤니티 소비형 사용자

직접 이미지를 생성하지 않더라도 다른 사람의 창작물을 감상하고 반응하는 데 관심이 있다.

### 주요 니즈

* 인기 이미지 탐색
* 트렌드 확인
* 좋아요 및 반응 참여
* 흥미로운 창작자 발견

---

# 3. 사용자 흐름

## 3-1. 회원가입 및 로그인

1. 사용자가 서비스에 접속한다.
2. 이메일 또는 소셜 로그인을 선택한다.
3. 회원가입을 완료한다.
4. 메인 피드로 이동한다.

---

## 3-2. AI 이미지 생성

1. 사용자가 “이미지 생성” 메뉴로 이동한다.
2. 프롬프트를 입력한다.
3. 생성 옵션을 선택한다.

   * 스타일
   * 이미지 비율
   * 생성 개수
4. 생성 버튼을 클릭한다.
5. AI가 이미지를 생성한다.
6. 생성 결과를 확인한다.
7. 원하는 이미지를 갤러리에 저장한다.

---

## 3-3. 개인 갤러리 관리

1. 사용자가 “내 갤러리”로 이동한다.
2. 생성한 이미지 목록을 확인한다.
3. 이미지 상세 정보를 조회한다.
4. 공개 여부를 설정한다.
5. 이미지를 정렬하거나 탐색한다.

---

## 3-4. 커뮤니티 공유

1. 사용자가 이미지를 공개 상태로 변경한다.
2. 이미지가 커뮤니티 피드에 노출된다.
3. 다른 사용자가 이미지를 감상한다.
4. 좋아요를 남긴다.
5. 작성자는 반응을 확인한다.

---

## 3-5. 이미지 탐색

1. 사용자가 커뮤니티 피드에 접속한다.
2. 최신순 또는 인기순으로 이미지를 탐색한다.
3. 관심 있는 이미지를 클릭한다.
4. 상세 페이지를 확인한다.
5. 프롬프트 및 생성 옵션을 참고한다.

---
